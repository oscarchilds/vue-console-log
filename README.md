# vue-console-log
A easy-to-use vue component that prints the browser's console log to the web page.
# Usage
It's as easy as:
```
npm i vue-console-log
```
and then
```
<script setup>
import ConsoleLog from 'vue-console-log'
</script>

<template>
  ...
  <ConsoleLog />
  ...
</template>
```
